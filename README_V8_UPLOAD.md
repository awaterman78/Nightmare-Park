# Nightmare Park, Monster Royale v8 Playable Loop

This package is the browser playable lab build. It avoids Unity Cloud and WebGL loader issues completely.

## What is new

- Real battle loop rather than static test scene.
- Grave Goblin deployment anywhere in the player half.
- Energy regeneration and 2 energy deployment cost.
- Enemy waves across three battles.
- Candle Imp enemy added.
- Towers, cores, win and lose states.
- Permanent upgrade choices after wins.
- Local browser save progress.
- Root `index.html` points GitHub Pages straight to this v8 build.
