# Browser v9 Combat Lab

This prototype is the first proper gameplay test for Nightmare Park: Monster Royale.

Controls:

- Tap a card, then tap the lower half of the arena to deploy.
- Or drag a card into the lower half of the arena.
- Destroy the enemy core before they destroy yours.
- Win battles to pick upgrades.

Cards:

- Grave Goblin
- Bone Pack
- Crypt Bruiser
- Screech Bats

Enemy units:

- Shade Goblin
- Candle Imp
- Rot Hound
