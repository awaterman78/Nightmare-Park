# Arena One Prefab Setup

## Required parts

```text
ArenaOne
  ArenaBackground
  DeploymentPlane
  PlayerDeploymentVisual
  InvalidPlacementVisual
  BridgePoints
  Cores
  Towers
```

## Background

Use the approved Arena One image as a plane texture for the first Unity test.

## Deployment plane

Use an invisible collider plane covering the arena. Raycasts hit this plane for placement.
