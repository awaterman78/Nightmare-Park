# Nightmare Park: Monster Royale v9 Combat Lab

This ZIP is ready to upload to the root of the GitHub repo.

## Upload

1. Unzip this file.
2. Open the `Nightmare-Park` repository on GitHub.
3. Upload the unzipped contents directly into the repo root.
4. Commit the changes.
5. Open the live GitHub Pages URL.

Main URL after upload:

`https://awaterman78.github.io/Nightmare-Park/`

Direct test URL:

`https://awaterman78.github.io/Nightmare-Park/prototypes/browser_v9_combat_lab/`

## What v9 adds

- Actual battle loop.
- Four playable monster cards.
- Tap or drag deployment.
- Energy cost and regeneration.
- Enemy AI deployment.
- Player and enemy towers.
- Player and enemy core health.
- Unit targeting, movement and combat.
- Ranged projectiles.
- Win and loss condition.
- Saved progression and upgrades.
- Mobile first layout.

This is still a browser combat lab, not the final Unity project. It is designed to test whether the core game is fun before rebuilding the same loop properly in Unity.
