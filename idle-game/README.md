# Nightmare Park

A portrait idle game built around an evolving horror empire. Each location fills with attractions as the player earns money, then gives way to a completely new scene.

## Current vertical slice

The first playable build includes three locations, milestone based scene reveals, tapping, purchasable attractions, passive income, offline earnings and automatic location transitions.

## Run it

```bash
npm install
npm run web
```

For iPhone or Android testing, run `npm start` and open the project in Expo Go.
