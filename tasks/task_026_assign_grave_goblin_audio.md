# Task 026, Assign Grave Goblin Audio

## Goal

Wire placeholder sounds to Grave Goblin events.

## Steps

1. Create `SO_SfxLibrary`.
2. Assign placeholder sounds.
3. Add `SfxPlayer` to battle scene.
4. Assign Grave Goblin sounds to `UnitAudioController`.
5. Confirm animation events trigger sounds.

## Done when

Grave Goblin spawn, steps, Bone Stab, hit and death all make placeholder sounds.
