# Task 033, GitHub Actions Build Pack

## Goal

Upload GitHub Actions Build Pack v1 and move build diagnostics away from Unity Cloud.

## Steps

1. Upload this pack to the repo.
2. Go to Settings > Pages.
3. Set Pages source to GitHub Actions.
4. Run `Deploy Browser Prototype`.
5. Test the Pages link.
6. Run `Unity WebGL Diagnostic Build`.
7. Capture the first error if it fails.

## Done when

GitHub Actions can either deploy the browser prototype or show a readable Unity build error.
