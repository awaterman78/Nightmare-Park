# Task 003, Grave Goblin Prefab Implementation

## Goal

Create the first playable Grave Goblin prefab.

## Steps

- Create `PF_GraveGoblin`.
- Add model placeholder if final model is not ready.
- Add Animator.
- Add Health.
- Add UnitController.
- Add MeleeAttack.
- Add collider.
- Add health bar anchor.

## Done when
