# Task 022, Grave Goblin Feel Integration

## Goal

Make Grave Goblin feel alive in Unity.

## Steps

1. Add GraveGoblinScuttleMotion to the model root.
2. Add ScuttleMotionBinder.
3. Add DamageFeedbackBinder.
4. Add BoneStabFeedback.
5. Replace or compare MeleeAttack with MeleeAttack_WithFeedback.
6. Add VFX anchors.
7. Test movement, hit and attack feel.

## Done when

Grave Goblin feels small, twitchy, alive and satisfying to deploy.
