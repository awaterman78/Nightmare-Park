# Task 001, Grave Goblin Vertical Slice

## Goal

Create the first proper playable test for Nightmare Park, Monster Royale.

## Scope

Arena One only.

Grave Goblin only.

Player can place Grave Goblin anywhere in own half.

Grave Goblin must move like a small, twitchy, living goblin.

He must scuttle, attack with Bone Stab, react when hit and die properly.

## Quality bar

This must feel like a premium real time mobile battler unit, not a sliding image or prototype placeholder.

## Required features

### Arena

Use Arena One.

Keep the background clean and readable.

No lane guides.

Use subtle placement feedback only when the card is selected.

### Placement

Tap Grave Goblin card.

Tap anywhere in player half.

Goblin spawns at that point.

If tapped outside allowed area, show clear feedback.

### Movement

Goblin chooses a target.

Goblin scuttles toward bridge and target.

Movement must feel alive.

### Combat

Goblin attacks nearest valid target.

First attack uses Bone Stab bonus.

Towers can damage the goblin.

Goblin can die.

### Feedback

Health bars.

Spawn puff.

Dust movement.

Bone Stab slash.

Hit reaction.

Death puff.

Basic sounds if available.

## Done when

The player enjoys placing Grave Goblin and watching him move and fight.
