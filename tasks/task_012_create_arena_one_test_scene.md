# Task 012, Create ArenaOne_Test Scene

## Goal

Create the first Unity battle scene.

## Steps

1. Create scene `ArenaOne_Test`.
2. Add ArenaRoot.
3. Add ArenaReferences.
4. Add ArenaPathRouter.
5. Add PlacementValidator.
6. Add bridge point transforms.
7. Add player and enemy core placeholders.
8. Add player and enemy tower placeholders.
9. Add deployment plane.
10. Add battle camera.

## Done when

The scene can run and shows the correct Arena One layout.
