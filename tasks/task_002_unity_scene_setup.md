# Task 002, Unity Scene Setup

## Goal

Create the first Unity scene for Arena One.

## Steps

- Create `ArenaOne_Test`.
- Add Arena One background.
- Set portrait camera.
- Add player and enemy core placeholders.
- Add two towers per side.
- Add basic canvas with top HUD and bottom card zone.
- Add deployment plane collider.

## Done when

The scene looks like the approved Arena One layout and can run in play mode.
