# Character asset slots

V18 is ready for proper character art. Future real assets should be placed under:

assets/characters/<character-id>/

Recommended files per character:

- portrait.png
- idle.png
- walk.png
- attack.png
- special.png
- death.png
- manifest.json

The active V18 build still uses procedural canvas rigs, but the IDs and data hooks are now ready for sprite-sheet replacement.
