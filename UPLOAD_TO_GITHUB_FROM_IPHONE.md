# Upload this package to GitHub from iPhone

## Simple route

1. Download and unzip this folder in the iPhone Files app.
2. Go to GitHub in Safari.
3. Create a new private repository called:

```text
nightmare_park_monster_royale
```

4. Upload the contents of this folder into the repo.
5. Commit the upload.

## Better route later

When you have access to a PC or Mac:

1. Install Git.
2. Install Git LFS.
3. Clone the repo.
4. Move this folder into the repo.
5. Commit and push.

## Important

This package is the production foundation. It is not a finished Unity project yet.

It includes:

- game design docs,
- Grave Goblin specs,
- artist and developer briefs,
- task list,
- art direction files,
- starter Unity script drafts,
- the current browser prototype.
