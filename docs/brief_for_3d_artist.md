# Brief for 3D Character Artist, Grave Goblin

We need the first production monster for Nightmare Park, Monster Royale.

## Deliverable

A stylised 3D Grave Goblin model, rigged and animated for Unity.

## Character

Small, nasty, fast melee goblin.

Must match the approved Grave Goblin concept direction in `art_direction/grave_goblin`.

## Required assets

```text
GraveGoblin.blend
GraveGoblin_Model.fbx
GraveGoblin_Textures
GG_Idle.fbx
GG_Spawn_Leap.fbx
GG_Scuttle_Run.fbx
GG_Bone_Stab.fbx
GG_Hit_Reaction.fbx
GG_Death.fbx
```

## Key animation note

The scuttle is the make-or-break animation.

It must not look like a normal run. It should be low, twitchy, goblin-like, irregular and alive.

## Style

Stylised 3D mobile game character, rich detail, strong silhouette, expressive face, readable from top-down camera.
