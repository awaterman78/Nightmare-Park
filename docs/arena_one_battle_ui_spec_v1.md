# Arena One Battle UI Spec v1

## Purpose

The battle UI must support fast mobile understanding.

The arena is the star.

## Top HUD

Left:

Enemy core health.

Centre:

Battle timer.

Right:

Player core health.

## Small stat row

Optional in first Unity test.

If used, show:

1. Screams.
2. Bones.
3. Chaos.

These must not dominate the arena.

## Arena area

No big tutorial box.

No lane markers.

No huge labels on towers.

Use small readable health bars.

## Deployment feedback

When Grave Goblin card is selected:

1. Card lifts or glows.
2. Player deployment zone subtly glows green.
3. Invalid areas stay unhighlighted.
4. Tapping invalid area gives a quick red pulse or small feedback.

## Bottom UI

Energy orb.

Energy bar.

Card hand.

For first test, only one active card:

```text
Grave Goblin
Cost 2
```

Other slots can be empty dark card slots.

## Card selection state

Selected card should:

1. Lift upward slightly.
2. Glow green.
3. Show placement zone.
4. Deselect after successful placement.

## Result panel

Simple first version:

Victory.

Defeat.

Fight again button.

No progression needed yet.
