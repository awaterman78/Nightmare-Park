# Unity First Real Test Plan

## Test name

Grave Goblin Greybox Test

## Minimum loop

1. Energy fills.
2. Player selects Grave Goblin.
3. Player taps own half.
4. Goblin spawns.
5. Goblin moves via bridge.
6. Enemy tower shoots.
7. Goblin attacks.
8. Health changes.
9. Core can be destroyed.

## Do not add yet

- More cards
- More monsters
- Shop
- Progression
- Multiplayer

## Quality rule

If Grave Goblin does not feel good, fix him before expanding the game.
