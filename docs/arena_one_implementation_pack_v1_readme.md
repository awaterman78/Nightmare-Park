# Arena One Implementation Pack v1

## Project

Nightmare Park, Monster Royale

## Purpose

This pack turns Arena One from concept art into a usable gameplay arena.

It defines camera framing, tower positions, core positions, deployment zones, bridge routing, scene hierarchy, UI layout, and implementation scripts.

## Build target

Scene name:

```text
ArenaOne_Test
```

## Core rule

Arena One must look premium, but gameplay readability comes first.

No lane markers.

No ugly debug overlays.

No giant instruction panels.

The player should understand where they can place monsters and what the battle objective is.

## Included in this pack

1. Arena One scene implementation spec.
2. Camera and coordinate guide.
3. Deployment zone specification.
4. Bridge routing rules.
5. Tower and core position guide.
6. Battle UI layout spec.
7. Unity scripts for arena config, pathing, placement and setup.
8. Task files for implementation.

## First playable target

Player selects Grave Goblin card.

Player places Grave Goblin anywhere in own half.

Grave Goblin moves through the bridge system and attacks enemy targets.

Enemy dummy spawner places enemy goblins in enemy half.

Towers and cores can be damaged.

## Important

This pack assumes the Unity Implementation Pack v2 and Grave Goblin Asset Pack v1 have already been uploaded.
