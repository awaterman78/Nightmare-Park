# Unity Implementation Pack v2

## Project

Nightmare Park, Monster Royale

## Purpose

This pack moves the project from pure design documentation toward an actual Unity implementation.

It contains Unity script drafts, architecture notes, prefab setup instructions and scene setup tasks for the first playable vertical slice:

**Arena One + Grave Goblin**

## What this pack adds

- Cleaner Unity script architecture.
- Unit data model.
- Deployment system.
- Energy system.
- Health and damage.
- Targeting logic.
- Tower attack logic.
- Grave Goblin movement/attack controller.
- UI binding drafts.
- Test scene setup instructions.
- Unity editor checklist.

## Important

This is source code and setup structure. It still needs to be opened and wired inside Unity Editor on a PC or Mac.

## First build target

```text
Scene: ArenaOne_Test
Playable card: Grave Goblin
Deployment: anywhere in player half
Enemy: dummy tower and optional dummy goblin spawner
Win test: enemy core can be damaged and destroyed
```

## Designer rule

Do not add monster two until Grave Goblin feels alive.
