# Combat Feel Pack v1

## Project

Nightmare Park, Monster Royale

## Purpose

This pack adds the first layer of game feel for the Unity vertical slice.

The aim is to make the battle feel punchy, readable and alive.

## Focus

Arena One

Grave Goblin

Tower fire

Damage feedback

Deployment feel

No new monsters yet.

## What this pack adds

- Grave Goblin scuttle motion helper.
- Deployment ring feedback.
- Floating damage numbers.
- Hit flash.
- Tower bolt projectile.
- Screen shake.
- Combat feedback service.
- Health bar world follow helper.
- First feel tuning guide.

## Design rule

Do not add more content until the first unit feels good.

A boring goblin with more monsters is still a boring game.
