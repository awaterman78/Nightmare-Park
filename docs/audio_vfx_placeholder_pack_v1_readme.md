# Audio VFX Placeholder Pack v1

## Project

Nightmare Park, Monster Royale

## Purpose

This pack gives the Unity vertical slice named placeholder audio and VFX hooks.

The sounds are not final. They are production placeholders so the game can be tested with feedback instead of silence.

## Important

These placeholder sounds are deliberately simple.

The point is to validate timing and feel. We replace them later with proper designed audio once the gameplay works.
