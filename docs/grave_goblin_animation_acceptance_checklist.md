# Grave Goblin Animation Acceptance Checklist

Use this checklist when reviewing the first animated Grave Goblin.

## Scale

| Check | Pass |
|---|---|
| Unit is small compared with towers |  |
| Face and dagger are still readable |  |
| Health bar does not dominate the unit |  |
| Multiple goblins would not clutter the arena |  |

## Idle

| Check | Pass |
|---|---|
| Hunched posture |  |
| Breathing or body motion |  |
| Eye, ear or head twitch |  |
| Looks impatient and alive |  |
| Does not stand like a human |  |

## Spawn leap

| Check | Pass |
|---|---|
| Appears with energy and personality |  |
| Landing has impact |  |
| Immediately transitions into scuttle |  |
| Deployment feels satisfying |  |

## Scuttle run

| Check | Pass |
|---|---|
| Low to ground |  |
| Fast and twitchy |  |
| Uneven side to side motion |  |
| Head bob and shoulder motion |  |
| Tiny dust or foot contact cues |  |
| Does not glide |  |
| Does not look like a human run |  |

## Bone Stab

| Check | Pass |
|---|---|
| Clear anticipation |  |
| Sharp lunge |  |
| Damage lands at the right moment |  |
| Green slash reads clearly |  |
| Target reaction is visible |  |
| Goblin recoils after impact |  |

## Hit reaction

| Check | Pass |
|---|---|
| Damage causes visible flinch |  |
| Reaction is quick |  |
| Goblin returns to action naturally |  |

## Death

| Check | Pass |
|---|---|
| Death is clear |  |
| Green puff or bone scatter included |  |
| Unit is removed cleanly |  |
| Animation is not too long |  |

## Overall feel

| Check | Pass |
|---|---|
| Feels like a living creature |  |
| Feels premium enough to build on |  |
| Fun to deploy repeatedly |  |
