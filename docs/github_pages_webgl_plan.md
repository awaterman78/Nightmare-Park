# GitHub Pages WebGL Plan

```text
builds/webgl/latest/
```

## GitHub Pages URL

```text
https://awaterman78.github.io/Nightmare-Park/builds/webgl/latest/
```

```text
index.html

TemplateData/
```

Upload all of it.

## Note

GitHub Pages is fine for early WebGL tests. If builds get too large later, use a dedicated host.
