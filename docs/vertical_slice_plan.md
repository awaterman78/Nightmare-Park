# Vertical Slice Plan

## Objective

Create one premium-feeling test scene where Grave Goblin can be deployed into Arena One and fight.

## Success criteria

The player should say:

"That little goblin feels alive. I want to deploy him again."

## Scope

### Included

- Arena One.
- Grave Goblin.
- Energy system.
- One card.
- Place anywhere in player half.
- Enemy dummy deployer.
- Towers and cores.
- Win/lose condition.
- Basic sound and VFX.

### Excluded

- Full multiplayer.
- Account system.
- Matchmaking.
- Shop.
- Progression.
- Full roster.
- Card upgrading.
- Live economy.

## Build stages

### Stage 1, Unity scene

- Portrait camera.
- Arena background.
- Core/tower positions.
- Basic UI.

### Stage 2, placeholder unit

- Capsule or sprite moving to tower.
- Placement anywhere in half.
- Targeting and attacking.

### Stage 3, Grave Goblin model

- Import rigged model.
- Animator controller.
- Idle, scuttle, attack, hit, death.

### Stage 4, polish

- Hit effects.
- Dust.
- Screen shake.
- Health bar.
- Card selection animation.
- Deployment flash.
- Sound effects.

### Stage 5, test

- iPhone playable build.
- Check size, movement, readability and feel.
