# Testing The Actual Game Properly

## Browser prototype

Useful now for quick iPhone feel checks.

## Actual game

The real game needs Unity on PC or Mac.

Flow:

1. Open repo in Unity.
2. Create `ArenaOne_Test`.
3. Wire the scripts and prefabs.
4. Press Play in Unity.
5. Build WebGL.
6. Upload WebGL output to GitHub Pages.
7. Test on iPhone.

## First Unity test success criteria

- Unity opens without compile errors.
- Scene runs in portrait.
- Grave Goblin card can be selected.
- Touch placement works.
- Grave Goblin spawns in player half.
- Towers damage units.
- Win and lose state can trigger.
