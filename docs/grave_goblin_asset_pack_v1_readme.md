# Grave Goblin Asset Pack v1

## Project

Nightmare Park, Monster Royale

## Purpose

This pack defines the first proper production monster asset.

It is focused entirely on making Grave Goblin feel like a real, living, Clash-quality mobile battle unit.

## Asset target

Create a fully usable Grave Goblin asset pipeline:

- 3D model
- Textures and materials
- Rig
- Animation clips
- Unity prefab
- VFX anchors
- Audio hooks
- Animation events
- Acceptance checks

## Quality target

The Grave Goblin must feel:

- Small
- Fast
- Twitchy
- Vicious
- Alive
- Readable
- Fun to deploy

If he slides, glides, floats or runs like a tiny human, he fails.
