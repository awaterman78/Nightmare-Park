# Battle UI Pack v1

## Project

Nightmare Park, Monster Royale

## Purpose

This pack defines the first real Unity battle interface for Arena One.

It covers:

1. Top core health HUD.
2. Battle timer.
3. Energy orb and energy bar.
4. Grave Goblin card.
5. Empty card slots.
6. Deployment feedback.
7. Floating combat text.
8. World health bars.
9. Result panel.
10. iPhone safe area handling.

## Design goal

The UI must feel premium, readable and mobile first.

The arena remains the star.

The UI should support the battle, not smother it.

## First playable UI target

Only one active card is required:

```text
Grave Goblin
Cost 2
Common
Quick Scuttle
Bone Stab
```

Other card slots can be empty until monster two exists.

## Main rule

No giant instruction boxes over the arena.

No ugly debug labels.

No clutter.

The player should instantly understand:

1. My core health.
2. Enemy core health.
3. How much energy I have.
4. Which card I can play.
5. Where I can place it.
